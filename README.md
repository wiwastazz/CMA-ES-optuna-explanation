# CMA-ES и Optuna: Алгоритм адаптации ковариацианной матрицы

Этот репозиторий содержит подробный разбор с визуализацией работы эволюционного алгоритма **CMA-ES (Covariance Matrix Adaptation Evolution Strategy)**.

В файле `CMA-ES_optuna_explanation.ipynb` пошагово рассматривается, как алгоритм ищет глобальный минимум функции Розенброка — классической тестовой задачи для оптимизации.

## Что внутри

- **Теоретическое объяснение**: назначение ковариационной матрицы, размера шага (`sigma`), этапов сэмплирования, отбора лучших точек и адаптации параметров.
- **Практическая реализация**: код на Python с использованием `numpy` и `matplotlib`.
- **Визуализация** каждого шага:
  - Инициализация центра и области поиска.
  - Генерация точек из нормального распределения.
  - Отбор лучших кандидатов.
  - Обновление среднего (сдвиг центра).
  - Адаптация ковариационной матрицы и размера шага.
